Pray Together

Get it on the App Store here.

Description

Pray Together is social media that allows users to post religious forum topics to be discussed. It allows users of different beliefs to openly discuss religious topics, post prayer requests, etc.

Highlights of the developing process:

  • Parse backend.
  • Parse backend with database.
  • Uses storyboard for UI and application design.
  • Uses JSQMessages framework for chat to resemble the appearance of Apple Messages Application.
  • Uses Table Views.
  • Uses Apple Push Notification.
  • Supports iPhone 6, iPhone 6+, iOS 8+.

Scrible Scrable

Get it on the App Store here.

Description

Scrible Scrable is an application that allows you to draw, paint, and add drawings and paintings to photos in your photo library. This application also allows you to take pictures.

Highlights of the developing process:

  • Use of 3 different subviews in mainview.
  • Draws lines and strokes using Quartz 2D.
  • Import photos from image picker.
  • Take photo and automatically import to program for editing.
  • Able to set brush size and opacity manually.
  • Custom RGB color selector.
  • Able to use multiple colors.
  • Uses storyboard for UI and application design.
  • Supports iPhone 6, iPhone 6+, iOS 8+.
